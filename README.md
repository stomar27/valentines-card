# ❤️ My Interactive Valentine's Card

A fun website where I ask my Valentine special questions! (Spoiler: They can't say No).

## ✨ Features
- **Slippery No Button**: The "No" button runs away when they try to click it.
- **Growing Yes Button**: The more they try to say No, the bigger the "Yes" gets!
- **AI Sweet Talk**: Uses Google Gemini to write cute romantic messages every time they click "Yes".
